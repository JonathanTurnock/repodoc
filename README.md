# RepoDoc

Repository documentation tools for VS Code.

## Development

- `npm install` — install dependencies
- Press `F5` in VS Code to launch an Extension Development Host with the extension loaded
- `npm run compile` — type-check, lint, and bundle
- `npm run watch` — rebuild on change (esbuild + tsc in parallel)
- `npm test` — run extension tests

## Commands

| Command | Description |
| --- | --- |
| `RepoDoc: Hello World` | Placeholder command to verify the extension is running |
